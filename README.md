<h3>
Cli tool to watch anime with polish subtitles mainly from
<a href="https://shinden.pl">shinden.pl</a>
</h3>



Required dependencies:
```
  curl grep sed fzf
```

To run it, copy script file into ~/.local/bin
and run with ani-cli-pl
